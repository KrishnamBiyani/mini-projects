# Web Development Projects Collection

This repository contains a collection of frontend and backend web development projects demonstrating various technologies including React, Node.js, MongoDB, and vanilla JavaScript.

## Projects Overview

### 1. URL Shortener (/URL_Shortener)
A full-stack backend application allowing users to shorten long URLs and track visit analytics.
- **Tech Stack:** Node.js, Express, MongoDB, EJS, JWT Authentication.
- **Features:** User Signup/Login, URL generation, redirection, and click tracking.

### 2. Social Media Feed (/newsocialmedia)
A React-based frontend prototype mimicking a social media feed.
- **Tech Stack:** React (Vite), Bootstrap, Context API.
- **Features:** Create posts, view feed, delete posts, and sidebar navigation.

### 3. Basic Todo App (/basic-todo)
A clean and functional task management application.
- **Tech Stack:** React (Vite), CSS Modules.
- **Features:** Add tasks with due dates, view list, and delete completed tasks.

### 4. Tindog (/Tindog)
A responsive landing page designed for a fictional dog dating application.
- **Tech Stack:** HTML5, CSS3, Bootstrap 5.
- **Features:** Responsive layout, pricing cards, and carousel testimonials.

### 5. JS Calculator (/JS Calc)
A functional calculator web application.
- **Tech Stack:** HTML, CSS, JavaScript.
- **Features:** Mathematical operations with a clean user interface.

### 6. Dice Game (/Dice game)
A simple browser-based game to settle scores.
- **Tech Stack:** HTML, CSS, JavaScript.
- **Features:** Generates random dice rolls for two players and declares a winner.

### 7. Personal Portfolio (/CSS - My Site)
A static personal portfolio website.
- **Tech Stack:** HTML, CSS.
- **Features:** Skills showcase, bio section, and contact information.

## How to Run

### Static Sites (HTML/CSS/JS)
Navigate to the specific folder (e.g., /Tindog, /JS Calc) and open the `index.html` file directly in your browser.

### React Apps
Navigate to the project folder (e.g., /basic-todo, /newsocialmedia) in your terminal and run:
1. `npm install`
2. `npm run dev`

### Node/Backend Apps
Navigate to the project folder (e.g., /URL_Shortener) in your terminal and run:
1. `npm install`
2. `npm start`
